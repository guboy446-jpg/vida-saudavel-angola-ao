# Vida Saudável Angola

Portal de saúde, alimentação e bem-estar — site estático mobile-first (HTML/CSS/JS puro), pronto para Netlify, Vercel ou GitHub Pages.

## Estrutura
- `index.html` — página inicial
- `pages/` — blog, categorias, ferramentas, páginas legais
- `articles/` — artigos publicados
- `css/style.css`, `js/script.js`
- `sitemap.xml`, `robots.txt`, `manifest.json`, `sw.js` — SEO e PWA

## Deploy
Site 100% estático — não precisa de build. Publicar diretamente a partir da raiz do repositório.
